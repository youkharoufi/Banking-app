# Banking-app
Bank App

Ceci est un guide d'utilisation de l'application Zsoft Bank :

1Pour que le projet se mette en route il faut entrer les commandes suivantes sur le terminal de Visual Studio Code :
- npm install
- npm i react-router-dom
- npm install react --save react-router-dom
- npm start

2/ Si vous n'entrez pas d'identifiants (nom ou prénom ou mot de passe vides) dans le formulaire d'accueil vous receverez une notification vous indiquant ce qui doit être fait.

3/ Si vous décidez de faire un dépôt ou un retrait, le montant entré doit être compris entre 10 Euros (valeur minimale) et 10000 Euros (valeur maximale). Si cette contrainte n'est pas respectée vous recevrez une nouvelle notification.

4/ Après avoir effectué un dépôt ou un retrait, vous pourrez accéder a votre solde via le bouton dédié.

5/ Chaque utilisateur débute avec 1000 Euros dans son compte (valeur par défaut).
